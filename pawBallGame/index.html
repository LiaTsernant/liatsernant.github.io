PawBall Game

About the game

PawBall is a tile-matching puzzle game.

The challenge is:
Chain of 200-500 balls of different colors start to roll around the screen.
Eliminate all of the balls before they make a circle and came back to the starting point.

User:
Has a paw on the middle of the screen with a ball of a random color.
When user sees two balls of the same color that match the pawâ€™s available color, user can shoot and remove this pair of balls from the board.
If user removes all balls before they make a circle and return to the starting point - user wins the game.
If user cannot remove all balls before they make a circle and return to the starting point - user lost the game.
